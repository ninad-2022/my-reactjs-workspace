Events:-
      An activity performed at a specific time is called an event. 

  There are two categories of events. 

  1. User events - The event triggred by the website user 
    Ex:- click, change, mousemove,etc. 

  2. Browser events - The events triggred by the web browser 
    Ex: load, online, offline, etc. 


  Event Handling:- 
    Performing an operation in response to an event called event handling. 


There are three ways to handle the events. 
1. By using event attributes 
    There are event attribute available on html elements 
    The event attribute names starts with 'on' followed by event name 
    Example: 
      onclick 
      Here the event name is 'click'. 

      You can pass javasccript code to the event attribute directly

      
      
2. By using event properties 
3. By using addEventListener()
