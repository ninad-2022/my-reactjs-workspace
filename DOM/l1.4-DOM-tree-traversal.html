DOM Tree Traversal
        Moving from one node to another is called tree traversal 
            
        - parent node
      
        - child node 
      
        - sibling node 
      
      
        Access parent node:- 
          - parentNode :- returns the parent node. 
          
          Accessing child node:- 
            - childNodes:- returns a list of all the children including element and 
            text nodes. 
      
          - children:- returns a list of element nodes by ignoring the text nodes. 
      
          - firstChild :-  returns the first child node including element and text 
      
          - firstElementChild:- returns the first element child node by ignoring text node 
      
          - lastChild:- returns the last child node including element and text nodes. 
      
          - lastElementChild:- returns the last element child node by ignoring the text node 
      
          - elementChildCount:- returns total number of element nodes 
      
          
      
          Accessing sibling nodes:- 
            - nextSibling:- returns the next sibling element including element and text 
      
            - nextElementSibling:- returns the next element sibling by ignoring text node 
      
            - previousSibling:- returns previous sibling node including element and text nodes 
      
            - previousElementSibling:- returns previous element sibling node by ignoring text node
